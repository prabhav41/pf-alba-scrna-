# pf-alba-scrna-
This pipeline analyzes **single-cell RNA sequencing (scRNA-seq)** data from the malaria parasite *Plasmodium falciparum* (NF54 strain) to visualize the expression of the **PfAlba protein family (Alba1–4)** across the parasite's intraerythrocytic life cycle stages (ring, trophozoite, schizont, gametocyte).
